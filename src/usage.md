*@grepbot help*
Prints the help message.
*@grepbot usage*
Prints the usage information.
*@grepbot list*
Lists the current greps that you have.
*@grepbot add <regex>*
Adds a new grep.
*@grepbot remove <regex>*
Removes a regex.
*@grepbot syntax*
Prints the help information for regex, and the rust regex engine.
